# Sprite Structure & Variables

## Sprites Used
- Player  
- Enemy (cloned)  
- Bullet (cloned)  
- Safe Zone  
- Health Bar or UI  
- Background  

## Key Variables
- Player HP  
- Enemy HP  
- Bullet Speed  
- Safe Zone Size  
- Alive Enemies  

## Scratch Lists
(Not mandatory, but recommended)
- Active Enemies  
- Damage Events  
- Spawn Points
